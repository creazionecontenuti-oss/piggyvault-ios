# Regola traduzione

## Obiettivo
Tradurre l'app in inglese, tedesco, spagnolo, italiano, francese, portoghese, cinese, giapponese, coreano, russo e hindi.

## Note
- Nelle opzioni dell app la lingua deve essere selezionabile con un menu con le bandierine e la lingua corrispondente.
- L'inglese deve essere la lingua di default.
- Di default la lingua selezionata deve essere quella del sistema. 
- Se la lingua del sistema non è supportata, l'inglese deve essere selezionata.
- Poi l'app deve essere tradotta in tutte le lingue menzionate sopra e in tutte le pagine. Se ti accorgi di qualche errore di traduzione, fai le tue correzioni. Se ti accourgi che qualcosa non è tradotta ed è hardcodato, fai le tue correzioni e rendi quella parte multilingua. Quindi facci attenzine ogni volta anche se nel prompt non c'è scritto.
